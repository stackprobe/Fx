======
Viewer
======


----
■インストール方法

アーカイブの中身をフォルダ構造・ファイル構成を維持したままローカルディスク上の任意の場所に展開して下さい。


----
■実行方法

Viewer.exe を実行して下さい。


----
■グラフのライン

薄いグレーのライン ... 安値と高値
緑のライン         ... 価格 (売値と買値の中間)
それ以外のライン   ... 移動平均


----
■操作方法

一番左上のドロップダウンリストで通貨ペアを選択します。

その下はグラフの開始日時 (グラフ左端の日時) を YYYYMMDDhhmmss 形式で入力します。

その下はグラフの終了日時 (グラフ右端の日時) を YYYYMMDDhhmmss 形式で入力します。

[<---] ボタン ... グラフの表示範囲を過去へスライドします。

[--->] ボタン ... グラフの表示範囲を未来へスライドします。

[拡大] ボタン ... グラフの右端を固定して拡大 (表示範囲を短く) します。

[縮小] ボタン ... グラフの右端を固定して縮小 (表示範囲を長く) します。

移動平均チェックボックス ... 移動平均の表示・非表示を切り替えます。
