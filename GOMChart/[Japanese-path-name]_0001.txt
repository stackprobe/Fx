----
フォーマット

通貨ペア:日時:インターバル:期間:(T|F):移動平均インターバル:移動平均_1:移動平均_2 ...

	日時 = YYYYMMDDhhmmss
	インターバル, 期間, 移動平均インターバル, 移動平均_x = 99d or 99h or 99m or 99
	高値安値を表示(T=する|F=しない)

	移動平均インターバル, 移動平均_x   ...   省略可能

----
操作

Enter     = グラフ更新
Alt  + 左 = 期間の(1/4)過去へ
Alt  + 右 = 期間の(1/4)未来へ
Ctrl + 左 = Y軸をキープして、期間の(1/4)過去へ
Ctrl + 右 = Y軸をキープして、期間の(1/4)未来へ
Ctrl + 上 = Y軸を狭める
Ctrl + 下 = Y軸を広げる
Roll-Up   = 期間を狭める
Roll-Down = 期間を広げる
